OpenID Connect

User Managed Access (UMA)

资源拥有者向授权服务器进行身份认证，可以自由选择: 用户名/密码、加密证书、安全令牌、联合单点登录或其它方式

授权服务器可以保存授权决策，后续可以跳过授权环节，沿用前一次的授权决策

也可以通过客户端白名单或者黑名单的内部策略来否决用户的决策
